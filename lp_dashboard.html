1 · Executive summary — key findings
Product UI is essentially invisible. Only 10 of 236 pages (4%) show product UI in the audit; 226 ask visitors to convert with zero in-page proof. The fix is template-level — and the practical scope is 194 pages (excluding 10 pricing pages and 22 API pages, which need different treatments — see G2 and G8).
Localization is a thin-content problem. 175 of 236 pages are non-EN (74%). Most are flagged "likely thinner than EN equivalent" — partial translations, not real localized assets. Localized homepages alone drive ~95k visitors.
21 competitor pages are mispositioned. All HIGH priority, all default to "cheaper than X" framing, all lack feature matrices and migration toolkits. ~22k visitors → only ~12 paid.
10 pricing pages globally underperform. All HIGH priority, all fail the A1 Owner persona. 110k visitors → 465 paid (0.42%). No agency tier highlighting, no Semrush/Ahrefs ROI, no migration credit.
AI-feature pages leak conversions. ChatGPT 1.3% reg rate, Gemini 1.2%, Perplexity 1.3% — 5–10× lower than non-AI feature pages. Visitors want methodology proof; pages don't supply it.
"easy-to-use" PPC cluster is anti-upmarket. 13 pages, 91k visitors, 7,582 regs — but only 28 paid. Attracts users who churn. Strategically harmful positioning.
API content is siloed. 22 technical/API pages exist but 27 high-traffic non-API pages flagged api=Yes barely link to them.
Solution pages are ROI-thin. All 22 (Agency / SMB / Entrepreneur / Enterprise) read like feature lists, not business cases. Agency page: 6,941 visitors → 2 paid.

5 · Quick wins vs strategic tasks

⚡ Quick wins (weeks 1–3)

  • 301-redirect 5 zombie "easy-to-use" locales — NL, NL/AC, IT, PT, JP. ~1h dev. Reclaims crawl + link equity.
  • Replace EN homepage hero copy — outcome-led headline + split signup CTA. Copy-only change on the biggest page.
  • Add "Switch from Semrush/Ahrefs free" ribbon to all 10 homepages. Single component, 10 placements.
  • Retitle competitor pages with use-case headlines. Copy-only — captures immediate SERP CTR gains.
  • Add "Most popular for Agencies" badge on /subscription.html Teams tier. One-line CMS change.
  • Add "Unlock full report" top bar on /google-ranking-checker.html and /seo-checker.html. First proof of G10.
  • Make Enterprise page demo-only — hide free-trial CTA on /se-ranking-for-enterprises.html.
  • Instrument tool-user pixel on all 10 free-tool pages. Unlocks high-intent retargeting audience.
  • Add API strip + link on 10 homepages and /subscription.html. ~11 placements, unblocks API traffic flow.

🏗 Strategic tasks (weeks 3–12+)

  • Build the 6-block component library — ProductUIStrip, ROI Strip, Migration Banner, Comparison Matrix, Methodology Box, Role-Split CTA. Underpins G1, G2, G3, G5, G6, G7.
  • Commission 3–5 named case studies per persona — Agency / SMB / Enterprise with named logos and quantified outcomes. Longest content lead time. Unblocks G5 + G7.
  • Build the Semrush → SE Ranking migration toolkit — guide + free data-import tool + 7-day consultation CTA. Unblocks G3.
  • Rework /api.html into endpoint-first hub — code samples, pricing per tier, endpoint catalog, comparison vs DataForSEO. Underpins G8.
  • Localization parity program — CMS checklist + backfill top 25 non-EN pages. Underpins G11.
  • Enterprise page rebuild — remove self-serve, expose SSO/SAML/SLA, named logo strip, sole demo CTA.
  • AI methodology documentation — publish methodology (LLMs · frequency · accuracy) as reusable Methodology Box. Highest-leverage AI-page fix.
  • Free-tool retargeting program — ad creative, audiences, dedicated trial LP. Unlocks the hidden value of G10.
